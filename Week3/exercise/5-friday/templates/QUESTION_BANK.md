# Technical question bank — Week 3 Java

Pick **≥3** per round (interviewer: vary difficulty).

1. Explain **encapsulation** vs **abstraction** with a tiny code or API example.
2. **Overload** vs **override** — how does the JVM pick which method runs?
3. Why must **`hashCode`** align with **`equals`** for **`HashMap`** keys?
4. When would you choose **`ArrayList`** over **`LinkedList`**?
5. **`HashMap`** vs **`Hashtable`** — what would you use today for a concurrent map?
6. What is a **functional interface**? Name two from `java.util.function.*`.
7. What does **`s -> s.length()`** compile to conceptually?
8. **Checked** vs **unchecked** exception — trade-offs for API design?
9. **`try-with-resources`** — what problem does it solve?
10. How does **`PriorityQueue`** ordering relate to **`Comparable`** / **`Comparator`**?

## Behavioral (STAR) — pick 1+

- Tell me about a bug you found before it reached production.
- Describe a time you had to learn a new tool quickly for an assignment.
- Tell me about a disagreement in a team project and how you handled it.
